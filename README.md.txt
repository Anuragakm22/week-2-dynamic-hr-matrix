# 🚀 SaaS Dashboard

A modern and responsive SaaS Dashboard built using **HTML, CSS, and JavaScript** as part of the **Week 2 Internship Task**. This project demonstrates dynamic DOM manipulation, event handling, form validation, and responsive dashboard design without using any external frameworks.

---

## 📌 Project Overview

The SaaS Dashboard allows users to manage records dynamically through a user-friendly interface. It includes analytics cards, user management, real-time dashboard counters, status management, and responsive layouts while showcasing core JavaScript concepts.

---

## ✨ Features

### 📊 Dashboard
- Responsive SaaS dashboard layout
- Sidebar navigation
- Sticky header
- Analytics overview cards
- Dynamic "Last Updated" date

### 👥 User Management
- Add users dynamically
- Form validation
- Role selection
- Department and phone details
- Join date selection

### 📈 Dynamic Functionality
- Live dashboard counters
- Pending, Accepted and Rejected status management
- Delete users dynamically
- Toast notifications
- Empty state message when no users exist

### 🎨 User Interface
- Responsive design
- Modern card layout
- Status badges
- Hover effects
- Professional color scheme
